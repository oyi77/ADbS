# Quick Start Guide

## Installation

### Using Homebrew (macOS/Linux)
```bash
brew tap your-username/adbs
brew install adbs
```

### Using npm (Cross-platform)
```bash
npm install -g adbs-cli
```

### Manual Installation
Clone the repository and add `bin/` to your PATH.

## Initialization

Navigate to your project root and run:
```bash
adbs init
```

This will create the necessary configuration files and directories.

## Workflow Basics

1. **Explore**: Understand the problem.
   `adbs start explore`
2. **Plan**: Create a strategy.
   `adbs start plan`
3. **SDD**: Define requirements, design, and tasks.
   `adbs start requirements`
   `adbs start design`
   `adbs start tasks`
4. **Execute**: Implement the code.
   `adbs start execution`

See [USER_GUIDE.md](./user_guide.md) for full details.
